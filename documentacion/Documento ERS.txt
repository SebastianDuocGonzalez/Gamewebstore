El proyecto consiste en una plataforma E-Commerce de videojuegos ("GameWebStore") con una arquitectura Full Stack. El sistema permite la venta de productos, gestión de inventario y administración de usuarios, integrando una aplicación web, un backend robusto y una aplicación móvil Android.

1.2 Tecnologías (Stack Tecnológico)

Frontend Web: React JS (Single Page Application).

Backend: Java Spring Boot (API REST).

Base de Datos: PostgreSQL 18 (Producción en Render), H2 (Testing).

Móvil: Android Nativo (Java/Kotlin).

Infraestructura: Despliegue en Render.com.

1.3 Requisitos Funcionales (RF)

RF-01 Gestión de Usuarios: Registro, Inicio de Sesión (JWT), Cambio de Roles (Admin/Trabajador/Cliente).

RF-02 Gestión de Productos: CRUD completo (Crear, Leer, Actualizar, Borrar) de videojuegos/consolas.

RF-03 Gestión de Órdenes: Carrito de compras, Generación de orden, Cambio de estado de orden.

RF-04 Visualización: Catálogo de productos, Detalles de producto, Perfil de usuario, Noticias y Eventos.

1.4 Requisitos No Funcionales (RNF)

RNF-01 Disponibilidad: El sistema debe estar accesible 24/7 mediante hosting en la nube (Render).

RNF-02 Seguridad: Las contraseñas deben estar encriptadas (BCrypt) y los endpoints protegidos por roles.

RNF-03 Pruebas: El frontend debe contar con pruebas unitarias usando Karma y Jasmine.
